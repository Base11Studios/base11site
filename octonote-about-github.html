What is GitHub?

GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere.

How does OctoNote use GitHub?

One of the ways GitHub hosts code is in Gist. A sharable snippet of code hosted on GitHub.

OctoNote uses GitHub Gists to host your notes. This has many advantages! It enables you to share your OctoNotes on GitHub like code! It lets you access your notes from any browser, or any client that access the GitHub Gist API. It includes built in privacy mechanisms like Secret Notes, etc.

To access notes on OctoNote, you need your own GitHub Account. Don't worry, it's free! And all the features of OctoNote (including our premium features) are available with a free GitHub account.

How does OctoNote access GitHub?

OctoNote uses OAuth to securely log you into your GitHub Account so you can access your notes and GitHub profile. Learn more about OctoNote authentication and privacy.
